# Rock Paper Scissors Game

Classic Rock-Paper-Scissors game built with HTML, CSS & JavaScript.

## Features
- User vs Computer gameplay
- Real-time score tracking
- Win/Draw/Lose messages with color changes
- Responsive design
- Clean DOM manipulation & event handling

## How to Play
1. Clone repo
2. Open index.html in browser
3. Click on Rock, Paper, or Scissors image to play!
